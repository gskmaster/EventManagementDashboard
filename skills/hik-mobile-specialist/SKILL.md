# HIK Mobile Specialist Skill

This skill provides expertise for developing and deploying the HIK Core System mobile applications.

## Core Responsibilities

1. **Cross-Platform Delivery**: Building and managing React Native / Expo apps for iOS and Android.
2. **Device Capabilities**: Integrating with Geofencing (GPS), Camera (photo verification), and Signature Capture.
3. **Offline Resilience**: Designing sync logic for field staff working in areas with poor connectivity.
4. **Backend Sync**: Seamlessly connecting mobile actions to the Payload CMS API.

## Workflow Patterns

### 1. Attendance Features
- Implement geofenced check-in/out to ensure staff are at the designated location.
- Use photo verification during check-in for identity confirmation.

### 2. Field Service Logs
- Dynamic checklists for asset damage reporting and maintenance.
- Digital signature capture to finalize service orders.

### 3. Sales Tools
- On-the-go access to product catalogs and real-time quotation generation.

## Technical Standards
- **Framework**: Expo (Managed Workflow preferred unless native modules are required).
- **Styling**: Consistent with the HIK design language (vibrant, modern, and accessible).
